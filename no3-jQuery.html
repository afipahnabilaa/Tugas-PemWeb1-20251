FAQ Interaktif

1. Apa itu jQuery?
jQuery adalah library JavaScript yang mempermudah penulisan kode JavaScript.
2. Apa fungsi click()?
click() digunakan untuk menjalankan perintah ketika elemen diklik.
3. Apa kegunaan hover()?
hover() berfungsi untuk memberi efek saat mouse masuk dan keluar dari elemen.
4. Apa itu toggle()?
toggle() digunakan untuk menampilkan dan menyembunyikan elemen.
5. Mengapa belajar jQuery?
Karena jQuery membuat manipulasi halaman web menjadi lebih mudah dan cepat.
